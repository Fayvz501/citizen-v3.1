# citizen-v3.1 — 1:1 upload guide

Этот репозиторий подготовлен под **полную загрузку проекта 1:1 без урезаний**.

## Как загружать проект правильно

Загружай **распакованное содержимое проекта в корень репозитория**, а не zip-файл.

После загрузки в корне должны лежать исходные файлы проекта, например:

- `package.json`
- `package-lock.json` или `yarn.lock`
- `src/`
- `public/`
- `.env.example` (если есть)
- `README.md` (если есть)
- любые остальные папки проекта

## Чего делать не нужно

- не загружай `.zip` как один файл
- не клади весь проект во вложенную папку
- не меняй структуру проекта перед загрузкой

## Для Render после 1:1 загрузки

Если `package.json` лежит в корне:

- **Root Directory**: пусто
- **Build Command**: `npm install`
- **Start Command**: `npm start`

## Environment Variables

Добавляй по `.env.example` проекта. Минимум часто нужны:

- `JWT_SECRET`
- `NODE_ENV=production`
